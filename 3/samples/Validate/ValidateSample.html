JSU - JS Utility Plugin - Validate Sample

JSU
SAMPLE DESCRIPTION
  • With default Validate Options, Items are instantly validated at onchange event
  • Clicking Submit button, all the items will be validated
  • Test other Validate Options with the OPTION Button
All the constraints described in validate.js are supported by in the first constraints parameter of cValidate(). In addition the constraints JSON parameter supports also the optional tip field, used to setup a Tooltip on the right of the item to validate.
With JSU Validate API only 2 calls are enought for BOTH constraints validation and UI Presentation (errors, Tips,..):
  • Setup everything (Constraints Validation, Error Presentation, Tooltips) with new cValidate()
  • In a button, or where you prefer, you can call cValidate.validateApply() to Validate all the section
of Sample_1:

cValidate constraints used in this Sample:
  • Email: Mandatory email
  • Duration: Mandatory duration in range [1..10] - Tooltip present
  • Birthday: Optional, if present it must be a date in format YYYY-MM-DD
  • Validate Options bOnErrShowSect=true is used to show a Section with all Validate Errors
  • Click Submit button to see the Error Section
  • Test other Validate Options with the OPTION Button
of Sample_2:

cValidate Options of This SAMPLE:
  • Validate all items with validateApply() method called when Submit Button is clicked
  • Error Section with all the Validate errors displayed on the Top
  • Alarm Gif in the Items with Validation error

cValidate constraints used in this Sample:
    • Username: Mandatory username with constraints explained in the tip
    • Password: Mandatory password with constraints explained in the tip
    • Confirm Password: Mandatory, must be equal to password
  • Validate Options bOnErrShowPopup=true is used to show a Popup with all Validate Errors
  • Click Submit button to see the Popup with Validate Errors
  • Test other Validate Options with the OPTION Button
of Sample_3:

cValidate Options of This SAMPLE:
  • Validate all items with validateApply() method called when Submit Button is clicked
  • Popup Error with all the Validate errors

cValidate constraints used in this Sample:
    • Size: Mandatory, the choice exclude XS or S - tip
    • Country: Mandatory, the choice must be within Italy or Spain - tip
    • ZIP CODE: Optional ZIP CODE with 5 digits if present
